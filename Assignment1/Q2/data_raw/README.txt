This file contains raw student performance data. The data does not reveal any identifying information about any students.
There are 8 columns in this dataset. Gender, race/ethnicity, parental level of education, lunch, test preparation course,
math score, reading score, and writing score. The possible values for each column are as follows. Gender is either "male"
or "female". race/ethnicity is split into 5 different groups, groups A, B, C, D, and E. The parents level of education can be
some high school, high school, some college, associate's degree, bachelor's degree, or master's degree. Lunch can either be 
standard, or free/reduced. Test preparation course can either be none or completed. Math score, reading score, and writing score
are all integers ranging from 0-100.
